🔌 Quiet Client API

API для подключения лоадера к сайту Quiet Client

POST /api/login

Проверка данных пользователя для лоадера

Параметры:

{
  "username": "string",  // Логин или email
  "password": "string"   // Пароль
}

Ответ при успехе:

{
  "success": true,
  "user": {
    "uid": 1,
    "username": "d1ago",
    "email": "d1ago@example.com",
    "subscription": {
      "active": true,
      "end_date": "2024-12-31T23:59:59.000Z",
      "status": "active"
    },
    "hwid": "ABC123DEF456",
    "role": "user",
    "registration_date": "2024-02-25T10:30:00.000Z"
  }
}

Ответ при ошибке:

{
  "success": false,
  "message": "Invalid credentials"
}

POST /api/update-hwid

Обновление HWID пользователя (привязка к ПК)

Параметры:

{
  "username": "string",
  "hwid": "string"
}

Ответ:

{
  "success": true,
  "message": "HWID updated successfully"
}

🧪 Тестирование API

📋 Инструкция для лоадера

Как подключить лоадер к сайту:

1. Проверка пользователя:

// JavaScript пример
const response = await fetch('https://your-site.com/api.html', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'login',
    username: 'd1ago',
    password: '123456'
  })
});

const data = await response.json();
if (data.success) {
  console.log('User UID:', data.user.uid);
  console.log('Subscription:', data.user.subscription.active);
  console.log('HWID:', data.user.hwid);
}

2. Обновление HWID:

// Когда лоадер запускается первый раз
const hwid = getComputerHWID(); // Ваша функция получения HWID
await fetch('https://your-site.com/api.html', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'update_hwid',
    username: 'd1ago',
    hwid: hwid
  })
});
